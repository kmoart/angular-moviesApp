# MoviesApp

## Dev

1. Clonar el proyecto o realizar la descarga desde el botón clone y luego hacer click en download ZIP
2. Instalar version de node js en tu pc , ir a la url de descarga: https://nodejs.org/en/download , luego hacer click en el botón fondo verde Windows Installer
3. Ejecutar descarga de nodejs y seguir instrucciones
4. Descomprimir descarga y abrir la carpeta donde se descargó el proyecto con el programa Visual Studio Code
5. Abrir una terminal tipo Git bash
6. Instalar angular con el comando : npm install -g @angular/cli
7. Instalar angular material con el comando : ng add @angular/material
8. Agregar el cdn de estilos que se encuentra en la siguiente web de Prime flex dentro de la etiqueta head del archivo principal de la aplicacion index.html:
  https://primeflex.org/installation  , buscar el cdn que tiene el titulo 'Load from CDN', copiarlo y pegarlo dentro de la etiqueta head del archivo index.html del proyecto.
10. Ejecutar ```npm install```
11. Ejecutar la app ```npm start``` o bien ```ng serve -o```
