# MoviesApp

## Dev

1. Clonar el proyecto o realizar la descarga desde el botón clone y luego hacer click en download ZIP
2. Descomprimir descarga y abrir la carpeta donde se descargó el proyecto con el programa Visual Studio Code
3. Abrir una terminal tipo Git bash
4. Ejecutar ```npm install```
5. Ejecutar la app ```npm start``` o bien ```ng serve -o```
